# Excel_Dashboard
This project demonstrates the practical use of Microsoft Excel for Data Analysis, covering Data Cleaning, Logical Functions, Sorting &amp; Filtering, VLOOKUP, INDEX-MATCH, Pivot Tables, Dashboards, and Data Visualization. It enhanced my skills in data handling, reporting, and generating insights using Excel tools.
